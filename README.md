# Akano 赤野 | 个人主页

一个现代化的个人主页网站，具有星空背景、流星动画和响应式设计。

## ✨ 功能特点

### 视觉效果
- 🌌 星空背景：200 颗随机分布的闪烁星星
- ☄️ 流星动画：随机划过的流星效果
- 🌫️ 高斯模糊：毛玻璃效果的 UI 组件
- 🎨 渐变设计：橙色主题的渐变色彩

### 功能模块
- ⏰ 时间天气浮窗：固定位置的实时时间、日期、农历和天气显示
- 📱 响应式导航：支持移动端汉堡菜单
- 🎭 滚动动画：内容区域滚动渐入效果
- 🌊 视差滚动：标题颜色随滚动变化

### 响应式设计
- 📱 移动端适配：支持 768px、480px、360px 多种屏幕尺寸
- 🔄 横屏优化：针对横屏模式的特殊优化
- 👆 触摸优化：针对触摸设备的交互优化

## 🎯 主要功能说明

### 1. 时间天气浮窗
- 位置：固定在页面右下角
- 功能：显示实时时间、公历日期、农历日期和天气
- 特性：高斯模糊背景、固定位置不可拖动、悬停上浮效果

### 2. 导航栏
- 桌面端：水平导航菜单
- 移动端：汉堡菜单（点击展开/收起）

### 3. 内容区域
- 关于我、我的兴趣、梦想座驾、工作室进程、联系我

## 🎨 技术栈

- HTML5：语义化标签
- CSS3：变量、Flexbox、Grid、媒体查询、动画
- JavaScript：Intersection Observer API、原生 DOM 操作

## 📱 响应式断点

| 断点 | 屏幕宽度 | 说明 |
|------|----------|------|
| Desktop | > 768px | 桌面端布局 |
| Tablet | ≤ 768px | 平板/大屏手机 |
| Mobile | ≤ 480px | 小屏手机 |
| Small Mobile | ≤ 360px | 超小屏幕 |
